# ⚡ X Auto-Poster Bot

Automatically posts tech/education tweets on X (Twitter) every ~10 minutes, 24/7.
No X API key needed — uses browser automation (Playwright).
Content generated by free Groq API.

## Topics
IITs • JEE • AI/ML • Study Abroad • Tech Careers • Coding Culture • EdTech • ISRO • Research

## Setup

### Space Secrets (Settings → Variables and secrets)
| Secret | Value |
|---|---|
| `GROQ_API_KEY` | Free key from [console.groq.com](https://console.groq.com) |
| `X_EMAIL` | X login email |
| `X_PASSWORD` | X password |
| `X_USERNAME` | X handle (without @) |

### Important
- **Use a spare X account** — bot accounts can get suspended
- Free tier Spaces sleep after ~48h — use UptimeRobot (free) pinging the URL
- Cookie-based session persists across restarts
- Fallback pool of 30+ pre-written tweets if Groq is down
